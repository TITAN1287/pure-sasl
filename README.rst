PureSASL
========
PureSASL is a pure python client-side SASL implementation.

At the moment, it supports the following mechanisms: ANONYMOUS, PLAIN,
CRAM-MD5, DIGEST-MD5, and GSSAPI. Support for other mechanisms may be
added in the future.
